1. Which of the following are operators, and which are values?

*
'hello'
-88.8
-
/
+
5

Answer:

> * - It is a Operator (Mutiples)

> 'hello' - It is a value

> -88.8 - It is a value

> - - It is a Operator (Substracts)

> / - It is a Operator (Divides)

> + - It is a Operator (Addition)

> 5 - It is a value
